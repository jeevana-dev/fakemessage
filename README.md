# 🛡️ Myth Buster

A machine learning project that detects fraudulent SMS messages using Logistic Regression and TF-IDF.

## Features

- Detect Fraud Messages
- Detect Legitimate Messages
- Confidence Score
- Built with Streamlit

## Technologies

- Python
- Streamlit
- Scikit-Learn
- TF-IDF
- Logistic Regression

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
